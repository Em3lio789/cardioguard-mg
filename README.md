## ❓ Troubleshooting

### Device Won't Connect

- Confirm your Holter device is powered and ready.  
- Try plugging into a different USB port or reconnect Bluetooth.  
- Restart CardioGuard and try again.  
- Check Windows Device Manager for driver issues.

### App Freezes or Crashes

- Restart the program.  
- Close other heavy programs running in the background.  
- Reboot your PC if problems persist.  
- Make sure you use the latest CardioGuard version.

### No Data or No Alerts

- Verify the device connection.  
- Confirm monitoring is started in the app.  
- Check that your device supports continuous data streaming with CardioGuard.
